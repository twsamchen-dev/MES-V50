MES 資料彙整中心－四頁串接說明 1. 將 MES_資料彙整中心.html 放到 GitHub Pages 與以下四份網頁相同資料夾： - index.html（Key In） - Target.html（幹部目標） - dashboard.html（戰情室） - aps.html（APS） 2. 導覽連結已預設完成，不需改網址。 3. 各頁可讀取本頁共用資料： const data = JSON.parse(localStorage.getItem('MES_DATA_CENTER_V1') || '{}'); 4. 接收訂單／料號／命令單查詢參數： const qs = new URLSearchParams(location.search); const orderNo = qs.get('order'); const itemNo = qs.get('item'); const commandNo = qs.get('command'); 5. 建議在四頁載入完成後： - 若 orderNo 有值，自動填入搜尋欄並執行搜尋。 - 若 itemNo 有值，自動套用料號篩選。 - 若 commandNo 有值，APS 自動定位該命令單。 6. 共用資料鍵名： MES_DATA_CENTER_V1 7. 大型 Excel 更新方式： 原始檔最大約 85 MB，不建議在手機或瀏覽器端即時解析。 建議每次更新 Excel 後重新產生本 HTML 內的彙整快照。
