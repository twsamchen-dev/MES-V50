與現有四份網頁互相連結

本頁導覽預設連到同一資料夾內的 index.html、Target.html、dashboard.html、aps.html。表格中的連結會帶入 ?order=訂單號&item=料號，既有網頁可讀取查詢參數後自動搜尋。

共用資料已寫入瀏覽器 localStorage，鍵名為 MES_DATA_CENTER_V1。其他四頁可用：

const data = JSON.parse(localStorage.getItem('MES_DATA_CENTER_V1') || '{}');
const qs = new URLSearchParams(location.search);
const orderNo = qs.get('order');
const itemNo = qs.get('item');
